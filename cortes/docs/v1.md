# Union Clips AI — Especificação Técnica

## Visão Geral

Sistema hiperpersonalizado de cortes automatizados para o Union Football Live. Recebe um link do YouTube, transcreve o conteúdo, usa Claude API para identificar os melhores momentos e gerar storytelling, e produz clips prontos com transições e logo via FFmpeg — tudo sem UI de edição de vídeo.

---

### 1. Módulo de Ingestão (`ingest/`)

```python
# ingest/downloader.py
import yt_dlp
import subprocess
from pathlib import Path

class YouTubeDownloader:
    """Baixa vídeo do YouTube e extrai áudio."""
    
    def __init__(self, output_dir: str = "/tmp/union-clips"):
        self.output_dir = Path(output_dir)
        self.output_dir.mkdir(parents=True, exist_ok=True)
    
    def download(self, url: str, video_id: str) -> dict:
        """Baixa vídeo completo + extrai áudio separado."""
        video_path = self.output_dir / f"{video_id}.mp4"
        audio_path = self.output_dir / f"{video_id}.wav"
        
        # Download com yt-dlp (melhor formato disponível)
        ydl_opts = {
            'format': 'bestvideo[height<=1080]+bestaudio/best[height<=1080]',
            'outtmpl': str(video_path),
            'merge_output_format': 'mp4',
        }
        with yt_dlp.YoutubeDL(ydl_opts) as ydl:
            info = ydl.extract_info(url, download=True)
        
        # Extrair áudio em WAV 16kHz mono (ideal para Whisper)
        subprocess.run([
            'ffmpeg', '-i', str(video_path),
            '-ar', '16000', '-ac', '1', '-f', 'wav',
            str(audio_path)
        ], check=True)
        
        return {
            "video_path": str(video_path),
            "audio_path": str(audio_path),
            "title": info.get("title", ""),
            "duration": info.get("duration", 0),
            "description": info.get("description", ""),
        }
```

### 2. Módulo de Transcrição (`transcribe/`)

```python
# transcribe/whisper_engine.py
from faster_whisper import WhisperModel
from dataclasses import dataclass

@dataclass
class TranscriptSegment:
    start: float      # timestamp início (segundos)
    end: float        # timestamp fim (segundos)  
    text: str         # texto transcrito
    confidence: float # confiança da transcrição

class WhisperTranscriber:
    """Transcrição com timestamps usando faster-whisper."""
    
    def __init__(self, model_size: str = "medium", device: str = "cpu"):
        # medium = bom equilíbrio qualidade/velocidade para pt-BR
        # large-v3 = melhor qualidade mas mais lento
        self.model = WhisperModel(
            model_size, 
            device=device, 
            compute_type="int8"  # quantizado para CPU
        )
    
    def transcribe(self, audio_path: str) -> list[TranscriptSegment]:
        """Transcreve áudio com timestamps por segmento."""
        segments, info = self.model.transcribe(
            audio_path,
            language="pt",           # Português brasileiro
            beam_size=5,
            word_timestamps=True,    # Timestamps por palavra
            vad_filter=True,         # Remove silêncios
            vad_parameters=dict(
                min_silence_duration_ms=500,
                speech_pad_ms=200,
            ),
        )
        
        result = []
        for seg in segments:
            result.append(TranscriptSegment(
                start=seg.start,
                end=seg.end,
                text=seg.text.strip(),
                confidence=seg.avg_logprob,
            ))
        
        return result
    
    def format_for_claude(self, segments: list[TranscriptSegment]) -> str:
        """Formata transcrição com timestamps para análise do Claude."""
        lines = []
        for seg in segments:
            ts_start = self._format_time(seg.start)
            ts_end = self._format_time(seg.end)
            lines.append(f"[{ts_start} → {ts_end}] {seg.text}")
        return "\n".join(lines)
    
    @staticmethod
    def _format_time(seconds: float) -> str:
        h = int(seconds // 3600)
        m = int((seconds % 3600) // 60)
        s = int(seconds % 60)
        return f"{h:02d}:{m:02d}:{s:02d}"
```

### 3. Módulo de Inteligência — Claude API (`intelligence/`)

Este é o coração do sistema. Claude recebe a transcrição completa com timestamps e o contexto da live (times, rodada, etc.) e retorna um plano de cortes estruturado em JSON.

```python
# intelligence/clip_analyzer.py
import anthropic
import json
from typing import Optional

# Prompt do sistema — PERSONALIZADO para Union Football Live
SYSTEM_PROMPT = """Você é o diretor de cortes do Union Football Live, o programa 
de transmissão de futebol brasileiro mais zoeiro e técnico ao mesmo tempo.

Seu trabalho é analisar a transcrição de uma live e identificar os MELHORES 
momentos para viralizar como cortes nas redes sociais.

## Personalidade Union Football
- Tom descontraído mas com análise técnica profunda
- Humor ácido sobre arbitragem, VAR e dirigentes
- Paixão genuína pelo futebol brasileiro
- Referências a dados estatísticos e história do futebol
- O mascote Bolinha pode ser mencionado nos overlays

## Critérios para Seleção de Cortes (em ordem de prioridade)
1. **MOMENTO VIRAL** — Reação a gol, lance polêmico, piada espontânea
2. **ANÁLISE TÉCNICA** — Insight tático brilhante, comparação histórica
3. **DEBATE QUENTE** — Discussão sobre polêmica do jogo/campeonato
4. **STORYTELLING** — História contada com começo, meio e fim
5. **MOMENTO BOLINHA** — Interação com o mascote/IA

## Regras de Corte
- Duração ideal: 30s a 90s (máximo 2 minutos para storytelling)
- Cada corte deve ter GANCHO nos primeiros 3 segundos
- Incluir contexto suficiente antes do momento-chave
- Nunca cortar no meio de uma frase importante
- Margem de segurança: +2s antes e +2s depois dos timestamps

## Formato de Resposta
Responda EXCLUSIVAMENTE em JSON válido, sem markdown, sem explicação."""

ANALYSIS_PROMPT = """Analise a transcrição abaixo da live "{title}" e identifique 
os melhores cortes.

{context}

## Transcrição Completa:
{transcript}

---

Retorne um JSON com a seguinte estrutura:
{{
  "clips": [
    {{
      "id": "clip_01",
      "title": "Título viral para o corte (max 60 chars)",
      "hook": "Frase de gancho para os 3 primeiros segundos",
      "category": "viral|analise|debate|storytelling|bolinha",
      "start_time": 1234.5,
      "end_time": 1289.0,
      "priority": 1,
      "storytelling": {{
        "setup": "Contexto breve do que estava acontecendo",
        "climax": "O momento principal do corte",
        "payoff": "Por que esse corte funciona"
      }},
      "production": {{
        "text_overlays": [
          {{
            "text": "Texto para overlay",
            "timestamp": 1240.0,
            "duration": 3.0,
            "position": "top|bottom|center",
            "style": "highlight|subtitle|meme"
          }}
        ],
        "transition_in": "fade|cut|zoom",
        "transition_out": "fade|cut|zoom",
        "suggested_thumbnail_time": 1245.0,
        "energy_level": "high|medium|low"
      }},
      "social_media": {{
        "caption": "Legenda sugerida para post",
        "hashtags": ["#UnionFootball", "#Brasileirão"],
        "best_platform": "reels|tiktok|shorts|twitter"
      }}
    }}
  ],
  "summary": "Resumo geral da live em 1-2 frases",
  "total_highlight_minutes": 0.0
}}"""


class ClipAnalyzer:
    """Usa Claude API para analisar transcrição e gerar plano de cortes."""
    
    def __init__(self, api_key: str):
        self.client = anthropic.Anthropic(api_key=api_key)
    
    def analyze(
        self,
        transcript: str,
        title: str,
        context: Optional[str] = None,
        max_clips: int = 10,
        model: str = "claude-haiku-4-5-20251001",  # Haiku para análise rápida e barata
    ) -> dict:
        """Analisa transcrição e retorna plano de cortes."""
        
        context_block = ""
        if context:
            context_block = f"""## Contexto da Live:
{context}"""
        
        user_prompt = ANALYSIS_PROMPT.format(
            title=title,
            context=context_block,
            transcript=transcript,
        )
        
        # Se transcrição for muito longa, usar Sonnet com contexto maior
        if len(transcript) > 100_000:
            model = "claude-sonnet-4-6"
        
        response = self.client.messages.create(
            model=model,
            max_tokens=8192,
            system=SYSTEM_PROMPT,
            messages=[{"role": "user", "content": user_prompt}],
        )
        
        raw_text = response.content[0].text
        return json.loads(raw_text)
    
    def refine_clip(
        self,
        clip_data: dict,
        transcript_segment: str,
        feedback: Optional[str] = None,
    ) -> dict:
        """Refina um corte específico com storytelling mais elaborado."""
        
        prompt = f"""Refine este corte para o Union Football Live.

Dados atuais do corte:
{json.dumps(clip_data, ensure_ascii=False, indent=2)}

Transcrição do trecho:
{transcript_segment}

{"Feedback do editor: " + feedback if feedback else ""}

Melhore o storytelling, os text overlays e a legenda para redes sociais.
Retorne o JSON atualizado do clip (mesma estrutura)."""
        
        response = self.client.messages.create(
            model="claude-sonnet-4-6",  # Sonnet para refinamento criativo
            max_tokens=4096,
            system=SYSTEM_PROMPT,
            messages=[{"role": "user", "content": prompt}],
        )
        
        return json.loads(response.content[0].text)
```

### 4. Módulo de Produção — FFmpeg (`producer/`)

```python
# producer/clip_producer.py
import subprocess
import json
from pathlib import Path
from typing import Optional

class ClipProducer:
    """Produz clips usando FFmpeg com transições e overlays."""
    
    def __init__(
        self,
        logo_path: str = "assets/union_logo.png",
        font_path: str = "assets/fonts/Montserrat-Bold.ttf",
        output_dir: str = "/tmp/union-clips/output",
    ):
        self.logo_path = Path(logo_path)
        self.font_path = Path(font_path)
        self.output_dir = Path(output_dir)
        self.output_dir.mkdir(parents=True, exist_ok=True)
    
    def produce_clip(
        self,
        video_path: str,
        clip_data: dict,
        output_format: str = "mp4",
    ) -> str:
        """Produz um clip completo com overlays e transições."""
        
        clip_id = clip_data["id"]
        start = clip_data["start_time"]
        end = clip_data["end_time"]
        production = clip_data["production"]
        
        output_path = self.output_dir / f"{clip_id}.{output_format}"
        
        # Construir filtergraph complexo do FFmpeg
        filter_parts = []
        
        # 1. Cortar segmento
        duration = end - start
        
        # 2. Fade in/out
        trans_in = production.get("transition_in", "fade")
        trans_out = production.get("transition_out", "fade")
        
        fade_in = "fade=t=in:st=0:d=0.5" if trans_in == "fade" else ""
        fade_out = f"fade=t=out:st={duration - 0.5}:d=0.5" if trans_out == "fade" else ""
        
        video_filters = [f for f in [fade_in, fade_out] if f]
        
        # 3. Logo overlay (canto superior direito, 80px, semi-transparente)
        logo_filter = (
            f"[1:v]scale=80:-1,format=rgba,"
            f"colorchannelmixer=aa=0.7[logo];"
            f"[tmp][logo]overlay=W-w-20:20"
        )
        
        # 4. Text overlays
        text_filters = []
        for i, overlay in enumerate(production.get("text_overlays", [])):
            text = overlay["text"].replace("'", "\\'").replace(":", "\\:")
            ts = overlay["timestamp"] - start  # Relativo ao início do clip
            dur = overlay.get("duration", 3.0)
            pos = overlay.get("position", "bottom")
            style = overlay.get("style", "subtitle")
            
            # Posição Y
            y_pos = {
                "top": "50",
                "center": "(h-text_h)/2",
                "bottom": "h-text_h-80",
            }.get(pos, "h-text_h-80")
            
            # Estilo
            fontsize = {"highlight": 48, "subtitle": 36, "meme": 56}.get(style, 36)
            box_opacity = {"highlight": 0.8, "subtitle": 0.6, "meme": 0.9}.get(style, 0.6)
            
            text_filters.append(
                f"drawtext=fontfile='{self.font_path}':"
                f"text='{text}':"
                f"fontsize={fontsize}:fontcolor=white:"
                f"x=(w-text_w)/2:y={y_pos}:"
                f"box=1:boxcolor=black@{box_opacity}:boxborderw=10:"
                f"enable='between(t,{ts},{ts + dur})'"
            )
        
        # Montar filtergraph completo
        vf_parts = video_filters + text_filters
        
        if vf_parts:
            video_filter_chain = ",".join(vf_parts)
            filter_complex = (
                f"[0:v]{video_filter_chain}[tmp];"
                f"{logo_filter}"
            )
        else:
            filter_complex = (
                f"[0:v]null[tmp];"
                f"{logo_filter}"
            )
        
        # Audio fade
        af = f"afade=t=in:st=0:d=0.3,afade=t=out:st={duration - 0.3}:d=0.3"
        
        # Comando FFmpeg final
        cmd = [
            'ffmpeg', '-y',
            '-ss', str(start),
            '-t', str(duration),
            '-i', video_path,
            '-i', str(self.logo_path),
            '-filter_complex', filter_complex,
            '-af', af,
            '-c:v', 'libx264', '-preset', 'medium', '-crf', '23',
            '-c:a', 'aac', '-b:a', '128k',
            '-movflags', '+faststart',  # Otimizar para streaming
            str(output_path),
        ]
        
        subprocess.run(cmd, check=True, capture_output=True)
        
        return str(output_path)
    
    def produce_vertical(self, video_path: str, clip_data: dict) -> str:
        """Produz versão vertical (9:16) para Reels/TikTok/Shorts."""
        clip_id = clip_data["id"]
        start = clip_data["start_time"]
        end = clip_data["end_time"]
        duration = end - start
        
        output_path = self.output_dir / f"{clip_id}_vertical.mp4"
        
        # Crop centro + scale para 1080x1920
        filter_complex = (
            "[0:v]scale=1920:-1,crop=1080:1920:(iw-1080)/2:0,"
            f"fade=t=in:st=0:d=0.5,fade=t=out:st={duration-0.5}:d=0.5[tmp];"
            "[1:v]scale=60:-1,format=rgba,colorchannelmixer=aa=0.7[logo];"
            "[tmp][logo]overlay=W-w-15:15"
        )
        
        cmd = [
            'ffmpeg', '-y',
            '-ss', str(start), '-t', str(duration),
            '-i', video_path,
            '-i', str(self.logo_path),
            '-filter_complex', filter_complex,
            '-c:v', 'libx264', '-preset', 'medium', '-crf', '23',
            '-c:a', 'aac', '-b:a', '128k',
            '-movflags', '+faststart',
            str(output_path),
        ]
        
        subprocess.run(cmd, check=True, capture_output=True)
        return str(output_path)
    
    def generate_thumbnail(self, video_path: str, clip_data: dict) -> str:
        """Gera thumbnail no timestamp sugerido pelo Claude."""
        clip_id = clip_data["id"]
        thumb_time = clip_data["production"].get(
            "suggested_thumbnail_time", 
            clip_data["start_time"] + 5
        )
        
        output_path = self.output_dir / f"{clip_id}_thumb.jpg"
        
        cmd = [
            'ffmpeg', '-y',
            '-ss', str(thumb_time),
            '-i', video_path,
            '-vframes', '1',
            '-q:v', '2',
            str(output_path),
        ]
        
        subprocess.run(cmd, check=True, capture_output=True)
        return str(output_path)
```

### 5. API FastAPI (`api/`)

```python
# api/main.py
from fastapi import FastAPI, BackgroundTasks, HTTPException
from pydantic import BaseModel, HttpUrl
from typing import Optional
import uuid
import os

from ingest.downloader import YouTubeDownloader
from transcribe.whisper_engine import WhisperTranscriber
from intelligence.clip_analyzer import ClipAnalyzer
from producer.clip_producer import ClipProducer

app = FastAPI(title="Union Clips AI", version="1.0.0")

# Estado dos jobs (em produção, usar Redis ou Firestore)
jobs: dict = {}

class ClipRequest(BaseModel):
    youtube_url: HttpUrl
    title: Optional[str] = None
    context: Optional[str] = None  # Ex: "Palmeiras 2x1 Corinthians - Brasileirão Rodada 15"
    max_clips: int = 8
    formats: list[str] = ["horizontal", "vertical"]
    model: str = "claude-haiku-4-5-20251001"  # Modelo padrão para análise

class ClipStatus(BaseModel):
    job_id: str
    status: str  # queued | downloading | transcribing | analyzing | producing | done | error
    progress: float  # 0.0 a 1.0
    clips: Optional[list[dict]] = None
    error: Optional[str] = None


@app.post("/api/clips", response_model=ClipStatus)
async def create_clips(request: ClipRequest, background_tasks: BackgroundTasks):
    """Inicia pipeline de geração de cortes."""
    job_id = str(uuid.uuid4())[:8]
    jobs[job_id] = {"status": "queued", "progress": 0.0}
    
    background_tasks.add_task(
        process_pipeline, 
        job_id, 
        str(request.youtube_url),
        request.title,
        request.context,
        request.max_clips,
        request.formats,
        request.model,
    )
    
    return ClipStatus(job_id=job_id, status="queued", progress=0.0)


@app.get("/api/clips/{job_id}", response_model=ClipStatus)
async def get_status(job_id: str):
    """Consulta status de um job."""
    if job_id not in jobs:
        raise HTTPException(status_code=404, detail="Job não encontrado")
    
    job = jobs[job_id]
    return ClipStatus(
        job_id=job_id,
        status=job["status"],
        progress=job["progress"],
        clips=job.get("clips"),
        error=job.get("error"),
    )


async def process_pipeline(
    job_id: str,
    url: str,
    title: Optional[str],
    context: Optional[str],
    max_clips: int,
    formats: list[str],
    model: str,
):
    """Pipeline completo de processamento."""
    try:
        # 1. Download
        jobs[job_id] = {"status": "downloading", "progress": 0.1}
        downloader = YouTubeDownloader()
        video_id = job_id
        video_info = downloader.download(url, video_id)
        
        if not title:
            title = video_info["title"]
        
        # 2. Transcrição
        jobs[job_id] = {"status": "transcribing", "progress": 0.3}
        transcriber = WhisperTranscriber(model_size="medium")
        segments = transcriber.transcribe(video_info["audio_path"])
        transcript_text = transcriber.format_for_claude(segments)
        
        # 3. Análise com Claude
        jobs[job_id] = {"status": "analyzing", "progress": 0.5}
        analyzer = ClipAnalyzer(api_key=os.environ["ANTHROPIC_API_KEY"])
        analysis = analyzer.analyze(
            transcript=transcript_text,
            title=title,
            context=context,
            max_clips=max_clips,
            model=model,
        )
        
        # 4. Produção dos clips
        jobs[job_id] = {"status": "producing", "progress": 0.7}
        producer = ClipProducer()
        
        produced_clips = []
        total = len(analysis["clips"])
        
        for i, clip_data in enumerate(analysis["clips"]):
            clip_result = {"data": clip_data, "files": {}}
            
            # Horizontal
            if "horizontal" in formats:
                h_path = producer.produce_clip(video_info["video_path"], clip_data)
                clip_result["files"]["horizontal"] = h_path
            
            # Vertical (Reels/Shorts/TikTok)
            if "vertical" in formats:
                v_path = producer.produce_vertical(video_info["video_path"], clip_data)
                clip_result["files"]["vertical"] = v_path
            
            # Thumbnail
            thumb_path = producer.generate_thumbnail(video_info["video_path"], clip_data)
            clip_result["files"]["thumbnail"] = thumb_path
            
            produced_clips.append(clip_result)
            
            progress = 0.7 + (0.25 * (i + 1) / total)
            jobs[job_id]["progress"] = progress
        
        # 5. Finalizado
        jobs[job_id] = {
            "status": "done",
            "progress": 1.0,
            "clips": produced_clips,
            "summary": analysis.get("summary", ""),
        }
    
    except Exception as e:
        jobs[job_id] = {
            "status": "error",
            "progress": 0.0,
            "error": str(e),
        }
```

### Cloud Run Config

```yaml
# cloud-run-service.yaml
apiVersion: serving.knative.dev/v1
kind: Service
metadata:
  name: union-clips-ai
spec:
  template:
    metadata:
      annotations:
        autoscaling.knative.dev/maxScale: "3"
        run.googleapis.com/execution-environment: gen2
        run.googleapis.com/cpu-throttling: "false"  # CPU sempre ativo
    spec:
      containerConcurrency: 1  # 1 job por instância (CPU-intensivo)
      timeoutSeconds: 1800     # 30 min timeout para lives longas
      containers:
        - image: gcr.io/PROJECT_ID/union-clips-ai
          resources:
            limits:
              memory: "8Gi"    # Whisper precisa de RAM
              cpu: "4"         # 4 vCPUs para FFmpeg paralelo
          env:
            - name: ANTHROPIC_API_KEY
              valueFrom:
                secretKeyRef:
                  name: anthropic-api-key
                  key: latest
            - name: GCS_BUCKET
              value: "union-clips-output"
```

## Diferencial: Prompt Engineering Personalizado

O grande valor deste sistema está na **qualidade do prompt**. O system prompt é treinado para entender:

1. **O estilo Union Football** — tom, humor, referências
2. **Futebol brasileiro** — contexto de campeonatos, rivalidades, polêmicas
3. **Viralidade** — o que funciona em cada plataforma (Reels vs Shorts vs TikTok)
4. **Storytelling** — setup → climax → payoff para cada corte
5. **Produção** — sabe pedir overlays, transições e thumbnails que fazem sentido

Com o tempo, o prompt pode ser refinado com base nos cortes que performam melhor nas redes — criando um **flywheel de melhoria contínua**.
