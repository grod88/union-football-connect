## 1. Visão de Produto

### O que é
Um módulo integrado ao Union Football Live que transforma lives e vídeos do YouTube em cortes profissionais usando IA — com **revisão humana** antes da produção final.

### Fluxos Principais

```
FLUXO 1: CORTES DE LIVE (pós-transmissão)
  YouTube URL → Transcrição → Análise IA → REVISÃO HUMANA → Produção → Publicação

FLUXO 2: QUADRO "GRANDE MOMENTO" (pré-jogo / compilação)
  Vídeos de gols + Comentário gravado → Template "Grande Momento" → Produção

FLUXO 3: CORTE RÁPIDO (durante a live)
  Marca timestamp ao vivo → Depois processa com template escolhido
```

### O Loop Humano (Human-in-the-Loop)

```

## 2. Sistema de Templates

Cada template define um **layout visual** com zonas de vídeo, texto e efeitos.
Inspiração direta: "Gol — O Grande Momento do Futebol" (TV Cultura).

### Templates Disponíveis

```
TEMPLATE: reaction
─────────────────────────────
│                           │
│     VÍDEO PRINCIPAL       │
│     (live / comentário)   │
│                           │
│   [LOGO]     [TEXT OVERLAY]│
─────────────────────────────
Uso: Corte simples da live com logo e texto.
Ideal para: reações, análises curtas, momentos virais.


TEMPLATE: split_horizontal  
─────────────────────────────
│     VÍDEO COMENTÁRIO      │
│     (frame de cima)        │
├───────────────────────────┤
│     VÍDEO DO LANCE         │
│     (frame de baixo)       │
─────────────────────────────
Uso: Comentário em cima, lance/gol embaixo.
Ideal para: "reage ao gol", análise tática com replay.


TEMPLATE: split_vertical
──────────────┬──────────────
│  COMENTÁRIO │   LANCE     │
│  (esquerda) │  (direita)  │
│             │             │
──────────────┴──────────────
Uso: Lado a lado.
Ideal para: comparações, antes/depois.


TEMPLATE: pip_bottom_right (Picture-in-Picture)
─────────────────────────────
│                           │
│     VÍDEO PRINCIPAL       │
│                    ┌─────┐│
│                    │ PIP ││
│                    └─────┘│
─────────────────────────────
Uso: Vídeo principal com miniatura do outro.
Ideal para: narração sobre lance, destaque do Bolinha.


TEMPLATE: grande_momento
─────────────────────────────
│ ░░ GRANDE MOMENTO ░░░░░░ │  ← Banner animado estilo TV Cultura
├───────────────────────────┤
│                           │
│     VÍDEO DO GOL          │
│                           │
├───────────────────────────┤
│ 🎙️ COMENTÁRIO ÁUDIO      │  ← Waveform visual + foto do narrador
│ "Que golaço, meus amigos" │
─────────────────────────────
Uso: Quadro especial — compilação de gols com narração.
Ideal para: pré-jogo, melhores da rodada, top 5.


TEMPLATE: versus
─────────────────────────────
│   TIME A    VS    TIME B  │  ← Escudos + placar
├───────────────────────────┤
│                           │
│     VÍDEO DO LANCE        │
│                           │
├───────────────────────────┤
│ 📊 STAT BAR               │  ← Posse, finalizações, etc.
─────────────────────────────
Uso: Contexto de jogo com dados.
Ideal para: highlights com estatísticas.


TEMPLATE: stories_vertical (9:16)
┌─────────────┐
│  [LOGO]     │
│             │
│   VÍDEO     │
│  (cropped)  │
│             │
│ ┌─────────┐ │
│ │LEGENDA  │ │
│ │ANIMADA  │ │
│ └─────────┘ │
│  #hashtags  │
└─────────────┘
Uso: Formato vertical para Reels/Shorts/TikTok.
Ideal para: qualquer corte adaptado para mobile.
```

### Schema do Template (TypeScript)

```typescript
// domain/entities/ClipTemplate.ts

export type TemplateId =
  | 'reaction'
  | 'split_horizontal'
  | 'split_vertical'
  | 'pip_bottom_right'
  | 'grande_momento'
  | 'versus'
  | 'stories_vertical';

export type VideoZoneId = 'main' | 'secondary' | 'pip';

export interface VideoZone {
  id: VideoZoneId;
  label: string;               // "Comentário", "Lance do Gol"
  required: boolean;
  position: {
    x: string;                 // FFmpeg expression: "0", "W/2", "W-w-20"
    y: string;                 // "0", "H/2", "H-h-20"
    width: string;             // "iw", "iw/2", "320"
    height: string;            // "ih", "ih/2", "180"
  };
  cropMode: 'fit' | 'fill' | 'stretch';
}

export interface TextZone {
  id: string;
  label: string;
  position: 'top' | 'center' | 'bottom' | 'custom';
  style: 'title' | 'subtitle' | 'highlight' | 'meme' | 'stat';
  animated: boolean;
  fontSizeBase: number;
}

export interface ClipTemplate {
  id: TemplateId;
  name: string;
  description: string;
  aspectRatio: '16:9' | '9:16' | '1:1';
  videoZones: VideoZone[];
  textZones: TextZone[];
  hasLogo: boolean;
  logoPosition: { x: string; y: string; scale: number; opacity: number };
  transitions: {
    intro: 'fade' | 'slide_left' | 'zoom_in' | 'wipe' | 'none';
    outro: 'fade' | 'slide_right' | 'zoom_out' | 'wipe' | 'none';
    betweenZones?: 'fade' | 'cut';
  };
  ffmpegPreset: string;        // Nome do preset FFmpeg pré-montado
}
```

## 4. Database Schema (Supabase)

```sql
-- migrations/001_clips_tables.sql

-- Lives/vídeos fonte
CREATE TABLE video_sources (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  youtube_url TEXT NOT NULL,
  youtube_id TEXT,
  title TEXT NOT NULL,
  description TEXT,
  duration_seconds INTEGER,
  context TEXT,                        -- "Palmeiras 3x1 Santos - Rod 20"
  video_storage_path TEXT,             -- Supabase Storage path
  audio_storage_path TEXT,
  transcript_json JSONB,               -- Array de TranscriptSegment
  transcript_text TEXT,                -- Texto formatado para Claude
  status TEXT DEFAULT 'pending',       -- pending | downloading | transcribed | analyzed | error
  metadata JSONB,                      -- Dados extras do yt-dlp
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now()
);

-- Insights da IA (pré-aprovação)
CREATE TABLE clip_insights (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  video_source_id UUID REFERENCES video_sources(id) ON DELETE CASCADE,
  title TEXT NOT NULL,                 -- "GOLAÇO DO VEIGA"
  hook TEXT,                           -- Frase gancho dos 3 primeiros segundos
  category TEXT NOT NULL,              -- viral | analise | debate | storytelling | bolinha
  priority INTEGER DEFAULT 0,
  start_time FLOAT NOT NULL,           -- Timestamp início (segundos)
  end_time FLOAT NOT NULL,             -- Timestamp fim (segundos)
  storytelling JSONB,                  -- { setup, climax, payoff }
  production_hints JSONB,              -- Sugestões de overlay, transição, etc.
  social_metadata JSONB,               -- caption, hashtags, best_platform
  ai_reasoning TEXT,                   -- Explicação da IA sobre por que escolheu
  suggested_template TEXT DEFAULT 'reaction',
  status TEXT DEFAULT 'draft',         -- draft | approved | rejected | producing | done
  
  -- Human review
  editor_notes TEXT,                   -- Nota do Guru para a IA
  editor_template_override TEXT,       -- Template escolhido pelo editor
  editor_time_override JSONB,          -- { start_time, end_time } ajustado
  secondary_video_url TEXT,            -- URL do vídeo paralelo (gol, lance)
  secondary_video_path TEXT,           -- Storage path se upload direto
  
  -- Resultado da produção
  ai_model_used TEXT,                  -- claude-haiku-4-5-20251001
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now()
);

-- Clips produzidos (pós-render)
CREATE TABLE produced_clips (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  insight_id UUID REFERENCES clip_insights(id) ON DELETE CASCADE,
  video_source_id UUID REFERENCES video_sources(id) ON DELETE CASCADE,
  template_id TEXT NOT NULL,
  
  -- Arquivos finais
  horizontal_path TEXT,                -- 16:9
  vertical_path TEXT,                  -- 9:16
  thumbnail_path TEXT,
  
  -- Metadados de produção
  duration_seconds FLOAT,
  resolution TEXT,                     -- "1920x1080"
  file_size_bytes BIGINT,
  ffmpeg_command TEXT,                 -- Comando usado (debug)
  
  -- Publicação
  published_platforms JSONB,           -- [{ platform, url, published_at }]
  view_count INTEGER DEFAULT 0,
  
  render_time_seconds FLOAT,
  status TEXT DEFAULT 'rendering',     -- rendering | done | error
  error_message TEXT,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- Templates customizados (além dos built-in)
CREATE TABLE custom_templates (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  description TEXT,
  config JSONB NOT NULL,               -- ClipTemplate serializado
  preview_image_path TEXT,
  is_active BOOLEAN DEFAULT true,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- Índices
CREATE INDEX idx_insights_source ON clip_insights(video_source_id);
CREATE INDEX idx_insights_status ON clip_insights(status);
CREATE INDEX idx_produced_insight ON produced_clips(insight_id);
CREATE INDEX idx_produced_status ON produced_clips(status);

-- RLS (Row Level Security) — simplificado para single-user
ALTER TABLE video_sources ENABLE ROW LEVEL SECURITY;
ALTER TABLE clip_insights ENABLE ROW LEVEL SECURITY;
ALTER TABLE produced_clips ENABLE ROW LEVEL SECURITY;
```

### Processamento Pesado: Supabase Edge Functions vs Worker Externo

O ponto crítico é: **Vercel serverless tem limite de 60s** (plano free) ou 300s (Pro).
Transcrição + FFmpeg podem levar **5-20 minutos**.

**Solução: Worker em fly.io ou Railway (grátis/barato)**

```
Vercel API → enfileira job no Supabase →
  Worker (fly.io) poll jobs pendentes →
    Download → Transcrição → salva no Supabase →
  Vercel API chama Claude →
    Salva insights no Supabase →
  Worker renderiza FFmpeg →
    Upload para Supabase Storage →
  UI atualiza via Supabase Realtime
```

Alternativa: **se preferir tudo Google Cloud**, um único Cloud Run job (como spec anterior) conectando ao mesmo Supabase.

## 8. Exemplo: FFmpeg Filter para Template `split_horizontal`

```typescript
// adapters/video/templates/split-horizontal.ts

export function buildSplitHorizontalFilter(params: {
  mainDuration: number;
  hasSecondaryVideo: boolean;
  textOverlays: Array<{
    text: string;
    relativeTime: number;
    duration: number;
    position: string;
    style: string;
  }>;
  logoPath: string;
  outputWidth: number;   // 1920
  outputHeight: number;  // 1080
}): string {
  const { outputWidth: W, outputHeight: H, mainDuration: dur } = params;
  const halfH = H / 2;  // 540

  // Cada zona ocupa metade vertical
  // Input 0 = vídeo principal (comentário)
  // Input 1 = vídeo secundário (lance/gol)
  // Input 2 = logo PNG

  let filter = '';

  // Escalar ambos os vídeos para largura total, metade da altura
  filter += `[0:v]scale=${W}:${halfH}:force_original_aspect_ratio=increase,crop=${W}:${halfH}[top];`;
  filter += `[1:v]scale=${W}:${halfH}:force_original_aspect_ratio=increase,crop=${W}:${halfH}[bottom];`;

  // Empilhar verticalmente
  filter += `[top][bottom]vstack=inputs=2[stacked];`;

  // Linha divisória (2px branca)
  filter += `[stacked]drawbox=x=0:y=${halfH - 1}:w=${W}:h=2:color=white:t=fill[divided];`;

  // Fade in/out
  filter += `[divided]fade=t=in:st=0:d=0.5,fade=t=out:st=${dur - 0.5}:d=0.5[faded];`;

  // Text overlays
  let current = '[faded]';
  params.textOverlays.forEach((overlay, i) => {
    const next = i === params.textOverlays.length - 1 ? '[texted]' : `[t${i}]`;
    const escaped = overlay.text.replace(/'/g, "\\'").replace(/:/g, "\\:");
    const fontSize = overlay.style === 'title' ? 48 : overlay.style === 'meme' ? 56 : 36;
    const yPos = overlay.position === 'top' ? '30'
      : overlay.position === 'center' ? `(${H}-text_h)/2`
      : `${H}-text_h-30`;

    filter += `${current}drawtext=fontfile='/app/assets/fonts/Montserrat-Bold.ttf':` +
      `text='${escaped}':fontsize=${fontSize}:fontcolor=white:` +
      `x=(w-text_w)/2:y=${yPos}:` +
      `box=1:boxcolor=black@0.6:boxborderw=10:` +
      `enable='between(t,${overlay.relativeTime},${overlay.relativeTime + overlay.duration})'${next};`;
    current = next;
  });

  if (params.textOverlays.length === 0) {
    filter += `[faded]null[texted];`;
  }

  // Logo overlay
  filter += `[2:v]scale=80:-1,format=rgba,colorchannelmixer=aa=0.7[logo];`;
  filter += `[texted][logo]overlay=W-w-20:20[out]`;

  return filter;
}
```
