# Union Clips AI - Assets

## Estrutura de Arquivos

```
assets/
├── union_logo.png              # Logo PNG com transparência (512x512+)
├── fonts/
│   ├── Montserrat-Bold.ttf     # Títulos e overlays
│   └── Montserrat-Medium.ttf   # Subtitles
└── audio/
    ├── intro_sting.mp3         # Som de abertura (2s)
    ├── outro_sting.mp3         # Som de encerramento (2.5s)
    └── bg/                     # Músicas de fundo (royalty-free)
        ├── hype.mp3            # Para momentos explosivos
        ├── tense.mp3           # Para polêmicas e VAR
        ├── chill.mp3           # Para análises calmas
        ├── epic.mp3            # Para grandes momentos
        └── funny.mp3           # Para zoeiras e memes
```

## Onde obter os assets

### Logo
- Baixar do Supabase Storage: `union-logo/union_logo.png`

### Fontes
- **Montserrat**: https://fonts.google.com/specimen/Montserrat
- Baixar as variantes Bold e Medium em formato TTF

### Músicas (royalty-free)
- **Pixabay Music**: https://pixabay.com/music/ (100% grátis, sem atribuição)
- **Uppbeat**: https://uppbeat.io/ (free tier com atribuição)
- **YouTube Audio Library**: Grátis para uso no YouTube

### Intro/Outro Stings
- Criar jingles curtos de 2-3 segundos
- Ou usar sons de transição do Pixabay/Uppbeat

## Especificações técnicas

| Asset | Formato | Duração/Tamanho |
|-------|---------|-----------------|
| Logo | PNG transparente | 512x512px ou maior |
| Fontes | TTF | - |
| Intro sting | MP3 320kbps | ~2 segundos |
| Outro sting | MP3 320kbps | ~2.5 segundos |
| BG music | MP3 320kbps | 1-3 minutos (loopável) |
